# 墨客星球
墨客星球是墨客实验室的文档聚集地。墨客星球只收录科学技术相关的文档。
## 如何访问
[下载最新版](https://gitee.com/XmacsLabs/mogan/releases)的墨干编辑器，在软件中点击`帮助->墨客星球`就可以访问。

这里所有的文档都开启了：`编辑->首选项...->其他->Use UTF8 for CJK in TM format`(该选项在墨干V1.1.1及以上版本可用)，因此TM格式文档可以显示中文。
