如何在Debian中切换默认Python版本

首先查看当前系统中默认的Python版本：

Shell session inside TeXmacs pid = 1042144 

Shell] 
python --version

Python 3.7.7

Shell] 

        

      

    

    

      然后查看当前系统可用的Python有哪些：
    

    

      

        

          
Shell] 

          
ls /usr/bin/python[0-9].*[0-9]

        

      

    

    

      

        

          

            /usr/bin/python3.10  /usr/bin/python3.7  /usr/bin/python3.8 
            /usr/bin/python3.9
          

        

      

    

    

      

        

          Shell] 
          

        

      

    

    

      我们可以把以上的python版本都放入python命令的候补列表：
    

    

      

        sudo update-alternatives –install /usr/bin/python
        python /usr/bin/python3.8 2
      

    

    

      

        sudo update-alternatives –install /usr/bin/python
        python /usr/bin/python3.9 3
      

    

    

      

        sudo update-alternatives –install /usr/bin/python
        python /usr/bin/python3.10 4
      

    

    

      现在我们可以列出这个候补列表
    

    

      

        

          
Shell] 

          
update-alternatives --list python

        

      

    

    

      

        

          

            /usr/bin/python3.10
          

        

      

    

    

      

        

          

            /usr/bin/python3.7
          

        

      

    

    

      

        

          

            /usr/bin/python3.8
          

        

      

    

    

      

        

          

            /usr/bin/python3.9
          

        

      

    

    

      

        

          
Shell] 

          
python --version

        

      

    

    

      

        

          

            Python 3.10.5
          

        

      

    

    

      

        

          Shell] 
          

        

      

    

    

      现在默认的Python版本已经切换到Python
      3.10.5，说明上面最后的数字越大，优先级越高。
    

    

      我们也可以使用
    

    

      

        update-alternatives –config
        python
      

    

    

      在终端中以交互模式选择我们的默认Python版本。
