2.2Template copyright

Current style files (.ts files) are under GPL license. If the templates (.tm files)/new style files (.ts files) are built on top of current style files, they should also under GPL license?

Document preamble (in .tm file) can define some macro, does it need to be GPL as well?

2.3What templates are wanted?

  • si-unit, style file for standard science units. Can copy paste keep the original style (from customized style/macro in .ts file/ preamble).

  • texmacs to html. Rich text editors have many customized tags in html, it can be useful for content creators (e.g. zhihu, blogs, github page).

  • Law and other professional area documents.

  • Interactive session/ teaching slides and notes/ UML graph

  • College thesis templates are nonprofit.
