2Topic: How to create a pull request in Gitee?

https://gitee.com/XmacsLabs/planet/issues/I6JNE2

2.1Some simple preparations

  1. You already have a Gitee account.

  2. Open the terminal and run the following command:

    git –version

2.2Generate and add SSH public key

  1. We need to generate SSH key by run the following command:

    ssh-keygen -t ed25519 -C "xxxxx@xxxxx.com"

    # Generating public/private ed25519 key pair…

    * The xxxxx@xxxxx.com here is just the name of the generated SSH key, and does not restrict or require it to be specifically named as a certain mailbox.

    * Just press Enter three times, and the ssh key will be generated, and no password required.

  2. We can print SSH public key using the cat command:

    cat ~/.ssh/id_ed25519.pub

  3. Copy the SSH public key to your clipboard.

  4. In the upper-right corner of any page of Gitee, click your profile photo, then click Settings. In the side bar, click SSH public key.

  5. Paste your public key into the Key field. Click Add SSH key.

2.3Git clone

  1. Go to the Code page of "XmacsLabs/planet", click the Clone/Download button, switch to SSH, and copy it.

  2. Return to your home directory in The Terminal using the cd command.

  3. We need to clone the warehouse by run the following command: 

    git clone git@gitee.com:XmacsLabs/planet.git

    (If prompted, enter YES.)

2.4Git add

  • Open the planet folder by file manager, and paste the files to be added here.

  • Enter the directory in The Terminal using the cd command: 

    cd planet

  • We can see which changes have been staged using the git status command.

  • Use the git add command: 

    git add XmacsLabs/X202303.tm

2.5Git config

  1. Add your own username using the git config command: 

    git config --global user.name "墨客实验室机器人"

  2. Add your own E-mail using the git config command: 

    git config --global user.email "bot@xmacs.net"

  3. You can view which changes have been done using the cat command: 

    cat ~/.gitconfig

2.6Git commit

You can make a commit using the git commit -m command: 

git commit -m "2023年第三次研讨会"

2.7Git push

Note. I already have the developer permission of the repository "XmacsLabs/planet". That's why I can push the code to the main branch and generate a pull request on Gitee. If you do not have the developer permission of the repository, you have to learn how to fork a Gitee/Github project and then create the pull request via the Gitee/Github web.

You can make a push using the git push origin main command. The pull request will be generated automatically.
